# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability, please report it responsibly.

**Email:** luiz@flylabs.fun

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

I will acknowledge receipt within 48 hours and work on a fix promptly.

## Scope

- The web application at flylabs.fun
- Supabase RLS policies and database security
- Authentication flows (email/password, Google OAuth)
- API keys and environment variable handling

## Out of scope

- Third-party services (Supabase, Vercel, Google Analytics)
- Social engineering attacks
- Denial of service attacks
